HTML Favicon

A favicon is a small image displayed next to the page title in the browser tab.

How To Add a Favicon in HTML

You can use any image you like as your favicon. You can also create your own favicon on websites like https://www.favicon.cc.

Tip: A favicon is a small image, so it should be simple and have high contrast.

Example of a Favicon

To add a favicon to your website, either save your favicon image in the root directory of your web server, or create a folder in the root directory called images and save your favicon image inside it. A common name for a favicon file is favicon.ico.

Example

<!DOCTYPE html>
<html>
<head>
  <title>My Page Title</title>
  <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Now, save the index.html file and reload it in your browser. Your browser tab should now display your favicon image to the left of the page title.

Chapter Summary

  • Use the HTML <link> element to insert a favicon.
  • The favicon image is displayed to the left of the page title in the browser tab.
  • Most browsers support multiple favicon image formats such as ICO, PNG, JPEG, GIF, and SVG.
