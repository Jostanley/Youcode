HTML Headings

HTML headings are titles or subtitles that you want to display on a webpage.

HTML headings are defined with the <h1> to <h6> tags.

Example:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Explanation

<h1> defines the most important heading, while <h6> defines the least important heading.

Note: Browsers automatically add some white space (margin) before and after a heading.

Headings Are Important

Search engines use headings to index the structure and content of your web pages. Users often skim a page by its headings. It is important to use headings to show the document structure.

  • <h1> — Main heading
  • <h2> — Subheading
  • <h3> — Less important heading
  • ...and so on up to <h6>.

Note: Use HTML headings for headings only. Do not use them just to make text big or bold.

Bigger Headings

You can change the size of any heading with the style attribute using the CSS font-size property.

Example:

<h1 style="font-size:60px;">Heading 1</h1 >
