HTML Text Formatting

HTML contains several elements for defining text with a special meaning.

Example

This text is bold

This text is italic

This is subscript and superscript

HTML Formatting Elements

Formatting elements were designed to display special types of text:

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

HTML <b> and <strong> Elements

The HTML <b> element defines bold text, without any extra importance.

This text is bold

The HTML <strong> element defines text with strong importance. The content inside is typically displayed in bold.

This text is important!

HTML <i> and <em> Elements

The HTML <i> element defines a part of text in an alternate voice or mood.

Tip: The <i> tag is often used to indicate a technical term, phrase from another language, or thought.

This text is italic

The HTML <em> element defines emphasized text. Screen readers pronounce it with emphasis.

This text is emphasized

HTML <small> Element

The HTML <small> element defines smaller text:

This is some smaller text.

HTML <mark> Element

The HTML <mark> element defines highlighted text:

Do not forget to buy milk today.

HTML <del> Element

The HTML <del> element defines deleted text (usually shown with a line through it):

My favorite color is blue red.

HTML <ins> Element

The HTML <ins> element defines inserted (underlined) text:

My favorite color is blue red.

HTML <sub> Element

The HTML <sub> element defines subscript text, used in chemical formulas or mathematical expressions:

This is subscripted text.

Water formula: H2O

HTML <sup> Element

The HTML <sup> element defines superscript text, often used for footnotes:

This is superscripted text.

Example: WWW[1]
