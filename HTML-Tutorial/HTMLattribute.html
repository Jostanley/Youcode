HTML Attributes

HTML attributes provide additional information about HTML elements.

Features of HTML Attributes

  • All HTML elements can have attributes.
  • Attributes provide additional information about elements.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs like: name="value".

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

<a href="html-intro.html">Go to homepage</a>

The src Attribute

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image:

<img src="../image/apple.png.jpg">

There are two ways to specify the URL in the src attribute:

  1. Absolute URL: Links to an external image hosted on another website. 
    src="https://www.ejistan.com/image/apple.jpg"
    Note: External images may be copyrighted.
  2. Relative URL: Links to an image hosted within your own site. 
    src="apple.png"

Tip: It’s usually best to use relative URLs — they won’t break if you change domains.

The width and height Attributes

These specify the size of the image in pixels:

<img src="img_girl.jpg" width="500" height="600">

The alt Attribute

The alt attribute provides alternate text if an image cannot be displayed.

<img src="apple.png.jpg" alt="This is an apple logo">

If the image is missing:

<img src="apple-logo.jpg" alt="This is an apple logo">

The style Attribute

Used to apply inline CSS styles to an element.

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

Defines the language of a web page. It should always be added to the <html> tag.

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

It can also include a country code:

<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

The title Attribute

Adds extra information as a tooltip when you hover over an element.

<p title="This is the tooltip">This is a paragraph.</p>

Attribute Best Practices

Use Lowercase Attribute Names

HTML is not case-sensitive, but using lowercase keeps code clean and consistent.

Always Quote Attribute Values

Quotes are not strictly required, but recommended. Without them, spaces break your code.

Good:
<a href="https://www.ejistan.com/html/">Visit our HTML tutorial</a>
Bad:
<a href=https://www.ejistan.com/html/>Visit our HTML tutorial</a>

Single or Double Quotes?

Double quotes are most common, but single quotes can be used if the value contains double quotes:

<p title='Learning "HTML" course'>
<p title="Learning 'HTML' course">

Chapter Summary

  • All HTML elements can have attributes.
  • The href attribute defines link destinations.
  • The src attribute specifies image sources.
  • The width and height attributes define image size.
  • The alt attribute provides fallback text for images.
  • The style attribute adds inline CSS styles.
  • The lang attribute defines the page’s language.
  • The title attribute adds tooltip information.
