This section covers how to use colors, background images, and opacity to design visually appealing pages.

1. color

Defines the text color of an element.

p { color: blue; }

This text is blue.

2. background-color

Sets the background color of an element.

div { background-color: lightgreen; }
This box has a light green background.

3. background-image

Displays an image as the background.

body { background-image: url('https://picsum.photos/300/100'); }
This div uses a background image.

4. background-repeat

Controls if the image should repeat.

div { background-repeat: no-repeat; }
No repeating image.

5. background-position

Sets the starting position of the background image.

div { background-position: center; }
Image centered.

6. background-size

Controls how the image fits its container: cover, contain, or a size value.

div { background-size: cover; }
Image covers the box.

7. background-attachment

Specifies if the background image scrolls with the page or stays fixed.

body { background-attachment: fixed; }
Scroll the page to see the fixed effect.

8. opacity

Defines how transparent an element is (0 = invisible, 1 = solid).

div { opacity: 0.5; }
This box is 60% visible.
