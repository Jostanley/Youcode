Using Google Fonts and Icons

Introduction

Typography and icons play a major role in how professional a website looks. In this lesson, you’ll learn how to use Google Fonts to give your site beautiful text styles and how to include Google Icons (Material Symbols) for clean, consistent visuals.

By the end of this tutorial, you’ll know how to: 1. Add Google Fonts to your website 2. Use multiple fonts for different sections 3. Add and style Google Icons 4. Combine both for a polished professional design

1. Adding Google Fonts to Your Website

Google Fonts provides free, open-source fonts you can easily embed into your web projects. Let’s start with a basic example of adding a new font to your site.

Go to fonts.google.com and choose a font you like. For this tutorial, let’s use “Poppins”. 

<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;600&display=swap" rel="stylesheet">

Place this <link> inside your <head> section before your CSS. Then, set it as your site’s font: 

body {
  font-family: 'Poppins', sans-serif;
}

Now, your whole website will use the “Poppins” font. If the browser can’t load it, it will fall back to a normal sans-serif font.

2. Using Multiple Fonts

You can mix fonts to give different sections unique personalities. For example, use one font for headings and another for body text. Let’s use “Poppins” for headings and “Roboto” for paragraphs. 

<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@600&family=Roboto:wght@400&display=swap" rel="stylesheet">

h1, h2, h3 {
  font-family: 'Poppins', sans-serif;
}

p {
  font-family: 'Roboto', sans-serif;
}

Using two complementary fonts makes your text easier to read and more visually appealing. Avoid using too many fonts — two or three are usually enough for a clean design.

3. Adding Google Icons (Material Symbols)

Google also provides a set of free icons known as Material Symbols. They’re scalable, customizable, and easy to add to your page. Add this line inside your <head>: 

<link href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined" rel="stylesheet" />

Now, you can use an icon anywhere in your HTML like this: 

<span class="material-symbols-outlined">home</span>

This displays a home icon. You can replace “home” with any icon name from the Material Symbols library (like “search”, “favorite”, “menu”, or “settings”).

4. Styling Your Icons

Icons behave like text, so you can change their size, color, and alignment using CSS: 

.material-symbols-outlined {
  font-size: 40px;
  color: #007bff;
  vertical-align: middle;
}

You can also use hover effects to make icons interactive: 

.material-symbols-outlined:hover {
  color: #ff9800;
  cursor: pointer;
}

Try adding these icons to your navigation bar or footer to make your site more visually engaging.

5. Example: Using Fonts and Icons Together

Let’s combine what you’ve learned into a mini project — a navigation bar with Google Fonts and Icons. 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Google Fonts & Icons Demo</title>
  <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@600&family=Roboto:wght@400&family=Material+Symbols+Outlined" rel="stylesheet">
  <style>
    body {
      font-family: 'Roboto', sans-serif;
      margin: 0;
    }
    nav {
      background: #007bff;
      color: white;
      padding: 15px 25px;
      display: flex;
      align-items: center;
      justify-content: space-between;
    }
    .logo {
      font-family: 'Poppins', sans-serif;
      font-size: 22px;
      font-weight: 600;
    }
    .nav-links a {
      color: white;
      text-decoration: none;
      margin: 0 10px;
      font-size: 16px;
    }
    .nav-links a:hover {
      text-decoration: underline;
    }
    .material-symbols-outlined {
      font-size: 20px;
      vertical-align: middle;
      margin-right: 6px;
    }
  </style>
</head>
<body>
  <nav>
    <div class="logo"><span class="material-symbols-outlined">language</span>MySite</div>
    <div class="nav-links">
      <a href="#"><span class="material-symbols-outlined">home</span>Home</a>
      <a href="#"><span class="material-symbols-outlined">info</span>About</a>
      <a href="#"><span class="material-symbols-outlined">contact_mail</span>Contact</a>
    </div>
  </nav>
</body>
</html>

This example combines modern fonts and icons to create a polished interface that feels like a real web app. Try customizing the fonts, colors, and icons to match your site’s theme.

Conclusion

In this lesson, you learned how to elevate your web design using Google Fonts and Icons. These tools bring style, readability, and clarity to your site — and they’re free! Always ensure your chosen fonts and icons match your site’s tone and purpose. Simplicity and consistency make your design professional.
