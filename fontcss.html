Fonts and text properties control how your content looks â€” from font type to spacing and alignment.

1. font-family

Defines the font type to use for text.

p { font-family: 'Courier New', monospace; }
This paragraph uses the Courier New font.

2. font-size

Sets the size of the font.

p { font-size: 24px; }
This text is 24px large.

3. font-weight

Controls how thick or bold the text appears.

p { font-weight: bold; }
This text is bold.

4. font-style

Makes text normal, italic, or oblique.

p { font-style: italic; }
This text is italic.

5. font-variant

Converts text to small-caps format.

p { font-variant: small-caps; }
This text uses small-caps style.

6. font (shorthand)

Combine all font properties in one line.

p { font: italic bold 20px Georgia; }
This is shorthand for font properties.

7. line-height

Adjusts space between lines of text.

p { line-height: 2; }
This paragraph has extra spacing between lines.
The line-height makes reading easier.

8. letter-spacing

Adjusts the space between individual letters.

p { letter-spacing: 3px; }
This text has wider letter spacing.

9. word-spacing

Adjusts the space between words.

p { word-spacing: 10px; }
This text has more space between words.

10. text-align

Aligns text to left, right, center, or justify.

p { text-align: center; }
This text is centered.

11. text-decoration

Adds underline, overline, or line-through to text.

p { text-decoration: underline; }
This text is underlined.

12. text-transform

Changes text case: uppercase, lowercase, or capitalize.

p { text-transform: uppercase; }
This text is uppercase.

13. text-shadow

Adds shadow behind text.

p { text-shadow: 2px 2px 5px gray; }
This text has a shadow.

14. white-space

Controls how spaces and line breaks are handled.

p { white-space: nowrap; }
This text will not wrap even if the screen is narrow.

15. direction

Sets the text direction: left-to-right (ltr) or right-to-left (rtl).

p { direction: rtl; }
This text flows from right to left.
